# Website assets

Static assets for the Dashpipe landing page. The HTML entry point is [`../index.html`](../index.html) at the repo root (GitHub Pages serves it at `/`).

| Path | Purpose |
|------|---------|
| `css/style.css` | Landing page styles |
| `js/main.js` | Mobile navigation |
| `CNAME` | Custom domain (`dashpipe.io`) — copied to site root on deploy |

Setup: [docs/PAGES_SETUP.md](../docs/PAGES_SETUP.md)
